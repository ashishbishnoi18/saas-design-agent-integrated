# Frontend Implementer Scaffold

This stage converts an approved architecture and visual design plan into production-oriented frontend code.

Suggested stack for the prompt examples:

- React / Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui where appropriate
- Playwright screenshot verification
- axe or equivalent accessibility checks

The implementer should never silently change the diagnosis, selected structure, or first-viewport contract. Any unavoidable change should be emitted as a structured exception.
